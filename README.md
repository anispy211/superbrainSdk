# Superbrain SDK: The Cognitive Memory Fabric 🧠✨

🔥 **v5.1.0 — The "Soul Expansion" is LIVE**

Superbrain Fabric is no longer just "volatile RAM". We've transformed the distributed memory fabric into a **high-performance, self-reflecting cognitive architecture** for AI agents.

## 💎 The Soul of the Fabric (v5.1.0 Breakthroughs)

### 🕰️ Memory History API (Hippocampus Layer)
Full versioned lineage of every memory block. Every metadata transition, tag update, and liveliness decay event is recorded in a Raft-consistent FSM, providing 100% auditability for agent actions.

### 📉 Layered Cognitive Compression (LCC)
Achieve **11-38x token reduction** before data hits the wire:
1.  **Deterministic**: Prunes HTML/JSON boilerplate.
2.  **Semantic**: Source-side Jaccard dedup for near-duplicate writes.
3.  **Extractive**: Consolidates long contexts into LLM-optimized summaries.

### 🕸️ Cortical Mesh (Knowledge Graph)
Distributed relational memory. Memories are linked via explicit edges (**part-of**, **conflicts-with**, **supports**) for recursive graph traversals and relational discovery.

### 🪞 MIRROR Tier (Recursive Reflection)
A specialized cognitive tier using **Lyapunov-inspired stability**. It reinforces "stable" memories through reflection, ensuring critical agent reasonings are protected from automatic decay.
